Cyber Spider

Completed Mar 2016 | UCLA CS 32 Project 4

Description:

This project revolved around creating an effective and efficient program that could crawl through a large number of computer log data and detect known malicious entities. The program also has the ability to discover new malicious entities associated with the already known malicious entities. The user then has the option to purge all associations within the log data that contain known malicious entities, emulating a sort of virus or attack defense program.

Frackman

Completed Feb 2016 | UCLA CS 32 Project 3

Description:

This project required us to program a small game. In this game, the player has to navigate the Frackman around an oil field until he finds all the barrels of oil, while also avoiding Protestors who want to stop the Frackman. We were provided with a code skeleton containing all the necessary assets to display and run the game and were tasked with programming all the superclasses and subclasses of the characters and objects within the game. Completing this project gave me a deeper understanding of proper object-oriented programming and how a game is constructed.

Rage Against the Robots

Completed Jan 2016 | UCLA CS 32 Project 1

Description:

This project required us to program a small console game. In this game, a character (indicated by the '@') is placed on a square grid among a swarm of robots (indicated by an 'R' or a number corresponding to the number of robots on that space) and has to defeat all the robots in order to win. If a robot happens to land on the same space as the character, then the player loses.

Rocks and Pebbles

Completed Nov 2015 | UCLA CS 31 Project 5

Description:

This project is small console game coded in C++. It's a word guessing game in which the program chooses a secret word 4-6 letters long (from a pre-selected list of words) and the player has to guess the word correctly in order to win. With each incorrect guess, the game informs the player of the number of correctly placed letters (rocks) and number of misplaced letters (pebbles) within their guess, and asks them to try again.
