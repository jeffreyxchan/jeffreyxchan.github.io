Cyber Spider

Completed Mar 2016 | UCLA CS 32 Project 4

Description:

This project revolved around creating an attack detection program that could crawl through a large number of computer log data and detect malicious entities. The program also has the ability to discover new malicious entities that are associated with an already known set of malicious entities. The user then has the option to purge all associations within the log data that contain known malicious entities, emulating a sort of virus or attack defense program.
